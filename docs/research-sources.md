# 食品類 ECFA MVP 目前採用的官方資料來源（第一輪）

> 範圍：台灣出口至中國大陸、食品類產品、ECFA 初判 / MVP。
> 狀態：第一輪官方文件蒐集與 MVP 取用原則，後續會再補中國大陸端即時優惠稅率表與更細的稅則解釋資料。

## 1) 財政部關務署－原產地規則總頁
- 來源：財政部關務署
- URL：https://web.customs.gov.tw/singlehtml/715
- 用途：作為所有 FTA / ECFA 原產地文件入口頁。
- 我用它來確認 ECFA 相關三份核心文件的官方下載位置。

## 2) 海峽兩岸經濟合作架構協定（ECFA）原產地認定基準－中文版
- 來源：財政部關務署 / 原產地規則頁面之官方下載檔
- URL：https://web.customs.gov.tw/download/m232104
- 用途：定義 ECFA 原產地認定基準。
- 在 MVP 的角色：
  - 作為「規則引擎」的母法來源之一
  - 用來判斷完全取得、實質轉型、區域價值含量等原產地邏輯

## 3) 海峽兩岸經濟合作架構協定（ECFA）原產地貨品清單
- 來源：財政部關務署 / 原產地規則頁面之官方下載檔
- URL：https://web.customs.gov.tw/download/m231959
- 用途：判斷某一貨品是否屬 ECFA 原產地貨品清單範圍。
- 在 MVP 的角色：
  - 作為「是否在 ECFA 範圍內」的第一道過濾條件
  - 初步拿來比對食品類章節/品項是否列入清單

## 4) 海峽兩岸經濟合作架構協定（ECFA）統一規章含原產地證明書
- 來源：財政部關務署 / 原產地規則頁面之官方下載檔
- URL：https://web.customs.gov.tw/download/m231957
- 用途：涉及統一規章、原產地證明書與實務作業要求。
- 在 MVP 的角色：
  - 作為後續文件檢核模組依據
  - 日後可延伸為「還缺哪些文件」提示功能

## 5) 財政部關務署－稅則稅率查詢及諮詢電話（官方入口）
- 來源：財政部關務署
- URL：https://web.customs.gov.tw/link/3078
- 轉址至：https://portal.sw.nat.gov.tw/APGQ/XGC411
- 用途：官方稅則稅率查詢入口。
- 備註：該查詢系統對非瀏覽器自動請求有限制，今晚 MVP 階段我先把它列為官方稅則查詢入口來源；後續會在人工驗證 / 前端使用流程中接入。

## 本輪 MVP 的採用原則
1. 先以 **ECFA 原產地貨品清單** 判斷「是否在協議覆蓋範圍內」。
2. 再以 **ECFA 原產地認定基準** 判斷「是否可能符合原產地規則」。
3. 再用 **統一規章含原產地證明書** 補上實務文件要求。
4. 食品類測試品（珍珠奶茶、鳳梨酥、芋頭酥）先用示意 BOM 建模，不直接宣稱最終法定稅則，僅作 MVP 驗證用途。
5. 中國大陸端最終適用優惠稅率、年度調整稅率與海關解釋，後續補入第二輪資料來源，以避免今晚在不完整資料下過度承諾。

## 目前對食品類測試品的暫定處理方式
- 珍珠奶茶：先作為「食品/飲料類示意測試案例」，用於測試 BOM 解析、來源國與可調整原料邏輯。
- 佳德鳳梨酥：先作為烘焙糕點類測試案例。
- 阿聰師芋頭酥：先作為烘焙糕點類測試案例。

## 風險聲明
MVP 階段的目標是建立：
- ECFA 範圍初判
- BOM 調整模擬
- 優惠可能性排序

不是直接取代最終報關、海關法律認定或專業關務簽證。
